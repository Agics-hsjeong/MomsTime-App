# 👤 Mom's Time User Flow

> **User Journey & Task Flow v1.0**

---

# 1. 목적

본 문서는 Mom's Time 사용자가 서비스를 이용하면서 수행하는 주요 행동(Task)을 정의한다.

User Flow는 기능이 아니라 **사용자의 목표**를 중심으로 설계한다.

---

# 2. 핵심 사용자 목표

Mom's Time 사용자는 다음 목표를 달성하기 위해 앱을 사용한다.

1. 오늘 해야 할 건강 관리를 확인한다.
2. 약과 영양제를 제시간에 복용한다.
3. 건강 기록을 남긴다.
4. 검사 일정을 놓치지 않는다.
5. AI의 도움을 받아 건강한 습관을 유지한다.

# 20. User Flow Summary

Mom's Time의 모든 User Flow는

하나의 질문에서 시작한다.

> **"오늘 엄마가 건강한 하루를 보내기 위해 가장 먼저 해야 할 일은 무엇일까?"**

서비스는 그 답을 가장 쉽고 자연스럽게 제시해야 한다.

사용자는 고민하지 않고 행동하고,

행동은 건강한 습관으로 이어진다.

그것이 Mom's Time의 User Flow이다.
