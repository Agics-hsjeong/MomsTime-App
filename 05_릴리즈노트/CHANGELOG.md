## [0.3.0] - 2026-07-06 · 퍼블리싱 / 프로토타입

기획·디자인을 기반으로 **전체 화면 HTML 퍼블리싱과 클릭 가능한 프로토타입**을 완성했습니다.

### Added
- **전체 36화면 HTML 퍼블리싱** — 메인 15 + 서브 21 (서비스 소개 3P, 약 등록 4단계 위저드 포함)
- **디자인 시스템 스타일시트** `assets/css/common.css` — 컬러·타이포·라운드·그림자 토큰과 공통 컴포넌트(버튼·카드·칩·탭·바텀네비 등)
- **전용 SVG 일러스트 세트** `assets/js/illustrations.js` — AI 로봇 마스코트, 임산부, 브랜드 로고, 프리미엄 다이아몬드, 단계 선택 3종(준비·임신·출산 후)
- **모션 시스템** `assets/css/motion.css` + `assets/js/motion.js` — 스크롤/로드 순차 리빌, 버튼·탭 리플, 하단 탭 팝, 체크 애니메이션 (모션 최소화 설정 존중)
- **화면 간 내비게이션** — 행 전체 클릭(`data-href`, 내부 컨트롤 클릭 제외), 알림/CTA 링크화, 히스토리 없을 때 홈 폴백 처리
- **탭 뷰 구현** — 캘린더(달력/목록), AI 케어(케어/히스토리), 복약·검진·영양제 상세(각 4탭), 통계(6탭), 검사·검진(5탭), 건강 기록(5탭), 알림(카테고리 필터), 쿠폰(보유/사용/만료)
- **마이페이지 설정 4종** — 계정·보안, 언어·지역, 도움말·FAQ, 앱 정보
- **약 등록 4단계 위저드** — 기본 정보 → 복용 정보 → 알림 설정 → 확인
- **프리뷰 인덱스** `index.html` — 전체 화면 썸네일 그리드

### Changed
- 기능성 이모지 아이콘 약 200곳을 **Material Symbols 라인 아이콘**으로 교체하여 시안과 톤 통일
- 스플래시·로그인을 시안 구도(보태니컬 장면·앰비언트)로 재구성
- 가로 오버플로우 방지(`overflow-x: clip`) 전역 적용

### Fixed
- 모바일 프레임 우측 잘림(가로 오버플로우) 문제
- 칩 텍스트 세로 줄바꿈, 설정 리스트 제목·부제 겹침
