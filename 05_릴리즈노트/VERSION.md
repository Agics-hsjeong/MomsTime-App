## 개발 중 유지 규칙

1. **모든 병합(merge)** 시 `CHANGELOG.md` 의 `[Unreleased]` 에 항목 추가.
2. **버전 태깅** 시:
   - `[Unreleased]` → 새 버전 블록으로 이동 + 날짜 기입
   - `pubspec.yaml` 의 `version` 갱신 (`x.y.z+build`)
   - Git 태그 생성: `git tag v0.4.0`
   - 사용자 대상 변경이면 `RELEASE_NOTES.md` 에 사용자 언어로 추가
3. **BUILD 번호**는 스토어 업로드마다 무조건 +1 (버전이 같아도 올림).
